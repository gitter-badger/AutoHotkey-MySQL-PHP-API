Über die API

Mit der Api lässt sich eine MySQL Datenbank ganz einfach per Autohotkey verwalten und benutzen.

Beispielsweise können Sie den Rang ihres Fraktionsmitgliedes Online in einer Datenbank speichern und diesen Ingame abfragen. Um so z.B. Aktionen vom Rang des Spielers abhängig zu machen.

Die Api nutzt die Autohotkey Funktion "UrlDownloadToFile" um mit einem PHP Skript zu kommunizieren. Das PHP Skript baut eine Verbindung zur Datenbank auf und liefert Antworten bzw. führt Aktionen aus.

Funktionen lassen sich unabhängig voneinander verbieten bzw. erlauben. So behalten Sie die Kontrolle und beugen Missbrauch vor.

Vorraussetzungen

  • Webserver(PHP4+)
  • MySQL Datenbank
  • FTP Client

Installation

  1. Klicken Sie auf Download um die Api herunterzuladen.
  2. Entpacken Sie das Verzeichniss.
  3. Öffnen Sie die 'config.php' und tragen Sie ihre MySql Datenbank Zugangsdaten in die dazu vorgesehenen Variablen ein.
  4. Überprüfen Sie die Rechte und passen Sie diese ggf. an ihre Bedürfnisse an.
  5. Laden Sie nun die 'improv3d.php' und die "config.php" auf ihren php-fähigen Webserver.
  6. Öffnen Sie anschliessend die 'improv3d.ahk' und tragen Sie ihren Server mit Pfad zur 'improv3d.php' in die dafür vor gesehene Variable ein.
